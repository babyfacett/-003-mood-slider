# 03 今日の気分スライダー

このアプリでは、スライダーで1〜5の範囲から現在の気分を選択し、その値に応じたメッセージを表示します。数値が低いほど気分が悪い、高いほど気分が良いとみなします。

## 使い方

1. 下記コマンドでアプリを実行します。

    ```bash
    streamlit run app.py
    ```

2. スライダーを動かして今日の気分を1（最低）から5（最高）までの間で選択してください。

3. スライダーの値に応じたメッセージが自動的に表示されます。気分が悪いときは休むことをおすすめし、気分が良いときはその調子を応援します。

Streamlit の `slider` コンポーネントで値を取得し、条件分岐でメッセージを変えています。
